# Aarogya Path

Aarogya Path is an AI-powered health platform designed to provide medical report analysis, disease information, and personalized health assessments. Built with a modern tech stack, this application aims to empower users with accessible healthcare insights.

## Features
- **AI-Powered Report Analysis**: Upload and analyze medical reports using advanced AI (powered by Groq).
- **Interactive Health Assessment**: Start a free assessment to evaluate your current health status.
- **Disease Information**: Browse comprehensive information about various diseases.
- **Secure Authentication**: User sign-up, login, and secure data persistence using Supabase.
- **Interactive Provider Dashboard**: Access a dedicated dashboard for seamless healthcare management.
- **Premium 3D Designs**: Stunning interactive elements powered by Spline, with responsive animations via Framer Motion.

## Tech Stack
- **Frontend Framework**: React 19 + Vite
- **Styling**: Tailwind CSS, Class Variance Authority, Tailwind Merge
- **Icons & Animations**: Lucide React, Framer Motion
- **Database & Auth**: Supabase
- **AI Integration**: Groq SDK
- **3D Graphics**: @splinetool/react-spline

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone this repository to your local machine:
   ```bash
   git clone <repository-url>
   ```

2. Navigate into the project directory:
   ```bash
   cd aarogyapath
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Set up the environment variables:
   Create a `.env` file in the root directory and add your Supabase and Groq API keys as required.

### Running the App Locally
To start the React application in development mode:
```bash
npm run dev
```
Open [http://localhost:5173/](http://localhost:5173/) to view the application in your browser.
